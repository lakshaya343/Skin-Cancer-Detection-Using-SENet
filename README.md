# Skin Cancer Detection Using SENet

## Overview
This project implements deep learning based skin cancer classification using SENet (SE-ResNet50) architecture on the HAM10000 dermoscopy dataset.

## Dataset
HAM10000 dataset contains 10,000 dermoscopic skin lesion images categorized into 7 classes.

## Model
- SE-ResNet50 (SENet)
- Transfer Learning
- Implemented using PyTorch and timm library

## Dataset Split
- Training: 70%
- Validation: 15%
- Testing: 15%

## Evaluation Metrics
- Accuracy
- Precision
- Recall
- F1-Score
- Confusion Matrix
- Loss Curve

## Technologies Used
- Python
- PyTorch
- Google Colab
- timm
- scikit-learn
- matplotlib

## Outputs
The model classifies skin lesion images into 7 categories successfully.
